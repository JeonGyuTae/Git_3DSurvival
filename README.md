## 1. 프로젝트 소개

1인칭 기반 3D 게임프로젝트입니다. 무인도에 표류하여 자원을 수집하고 동물을 사냥해나가며 뗏목을 타고 탈출하는 컨셉입니다.
이 프로젝트에서는 다음과 같은 기능을 구현하였습니다.

 - 캐릭터 이동 구현 및 상호작용
 - 자원 수집 및 가공
 - 적과의 전투
 - NPC와의 대화 기능
 - Map 지도 활성화
 - 식사와 수분 관리
